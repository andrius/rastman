# CLAUDE.md

Fork/mirror (under `andrius/rastman`) of Mathieu Lemoine's `rastman` - a Ruby interface to the Asterisk Manager API (AMI). It connects on the manager port (5038 by default), sends Manager commands (async by name, synchronous with a `!` bang), and listens to server events. Not Andrius's original code. Last commit 2018-11-13 - dormant.

Stack: Ruby gem (`rastman.gemspec`), Docker. Reference only.
